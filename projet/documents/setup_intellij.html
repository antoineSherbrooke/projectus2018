1 Get the sources

  1. install subversion (svn)
    • get the package subversion for the commandline tool
  2. checkout the repository
svn checkout svn+ssh://YOURCIP@svn.gel.usherbrooke.ca/grp/svn/bretagne16/projet

The root of the project is the directory projet/code/projectus

Don't forget to get the file dbRes.conf and place it in projectus/conf. A copy of the file is at the root of the depot in the directory projet.

2 Setting IntelliJ

  • get the Scala plugin
  • Import project from existing structure > SBT
  • Project SDK (jdk 1.8 , New and find it if necessary)

2.1 compile run and test from intellij (without activator)

  • Run > Edit configurations > +
    1. a "compile"
      1. SBT Task
      2. Tasks : compile
      3. remove -XX:MaxPermSize=256M if present
      4. In before launch
        • remove Make (select and (-))
    2. a "Run"
      • same manner as "compile" but with Tasks : run
    3. a "sbt test" *
      • same manner as "compile but with Tasks : test
    4. a "all tests"
      1. JUnit
      2. All in package, single module, Use classpath of module : root
      3. JRE: <your jdk 1.8>
      4. In before launch
        • remove Make (select and (-))
        • add (+) > Run Another Configuration > "compile"
  • you could also make a "rebuild" task
    • same manner as the "compile" but with Tasks : clean compile

Explanation sbt test : currently if tests had never been compiled (a clean checkout of the project, cleaned project), you have to launch sbt test once to be able to use JUnit tests (else there is an error Empty Test Suite).

Once the server is running : it launchs compilation automatically if necessary (for example, a refresh of localhost:9000/projectus and a file has changed).

2.2 use svn with intellij

  • disable multiple prompt for password
    1. ~/.subversion/config > store-passwords = yes (comment the line store-passwords = no)
    2. Settings > Subversion
      • use commandline client
      • enable interactive mode

The first time you have a prompt put your cip and password and tick "save credentials".

ProjectUS

Welcome to ProjectUS

ProjectUS is a software project management application that uses the Scrum framework. It has been developed by the eXit research group from the Université de Sherbrooke. For more information on how to use ProjectUS, download the Capsule Scrum ProjectUS.

If you want to know more about the Scrum framework, read: Scrum Guides

First Use

The first time that you use ProjectUS, There is a tutorial which allows you to setup your project. From the creation of the team until start the first sprint.

  • Only the project administrator can assign members to a team.
  • Only the Product Owner can create a release and add entries to the backlog.
  • Only the Scrum Master can add task to the sprint and start the sprint.
  • Normal Sprint, all registered developers can take a task and log the time for this task.
  • Then go back to step corresponding of the current need.

FAQ

There is an Frequent asked questions. Inside ProjectUS, you can interact with ProjectUS Team to facilitate your work.
